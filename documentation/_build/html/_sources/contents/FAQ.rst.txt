FAQ
===


.. note::

  **My logging doesn't work?**

  You have to log using your Intranet Ldap credentials. Read below as well.


.. note::

  **I'm a PI but I cannot create a bill or process them?**

  You have to log first using your Intranet Ldap credentials. Then, if it is still not working, send an email to
  `Sébastien Weber`_ asking for extra credentials in order to create and process bills.



.. _Sébastien Weber: sebastien.weber@cemes.fr
